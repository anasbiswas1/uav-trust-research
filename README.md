# uav-trust-research

Trustworthiness evaluation of machine-learning detectors for UAV security
(network intrusion detection and GPS/GNSS attack detection): calibration,
conformal prediction, and selective prediction under attack-family shift.

University of Portsmouth Drone Lab. See the project brief for datasets and conventions.

## Layout
- `notebooks/`  numbered, self-contained notebooks
- `src/`        shared modules (trust metrics, data handling)
- `reports/`    result CSVs
- `figures/`    generated figures
- `data/`       datasets and model binaries (git-ignored)
